No self xss

Prevent self-xss attacks on your website

This script is designed to display a warning message that scares the user and informs him that it is dangerous to copy Javascript code and paste it into the console.

This way you can prevent self-xss attacks.

For more information on Self-XSS visit this article on wikipedia.
